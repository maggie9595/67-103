Rock-Paper-Scissors Game

A Javascript Exercise

Instructions

Click a button to choose what you want to 'throw', that is, choose either rock, paper or scissors. The computer will randomly choose a value as well. Remember that paper covers (wins over) rock, scissors cut (wins over) paper, and rock breaks (wins over) scissors. If you and the computer throw the same item, it's a draw and no one gets a point.

Score

  • Losses: 0
  • Draws: 0
